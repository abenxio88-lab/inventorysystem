## 📋 **QUICK START: NEW FEATURES**

### **Process a Return:**
1. Go to "🔄 Returns/RMA" tab
2. Click "➕ New Return"
3. Enter customer details
4. Select items being returned
5. Choose reason
6. Create return
7. Approve return
8. Refund processed

### **Process Trade-In:**
1. Go to "💱 Trade-Ins" tab
2. Click "➕ New Trade-In"
3. Enter customer & product details
4. Set condition (excellent/good/fair/poor)
5. Set trade-in value
6. Set credit amount
7. Create trade-in

### **Create Service Ticket:**
1. Go to "🔧 Service/Repair" tab
2. Click "➕ New Service Ticket"
3. Enter customer details
4. Enter device details (type, brand, model, serial)
5. Describe issue
6. Set estimated cost
7. Create ticket
8. Track status (Received → In Progress → Completed)
